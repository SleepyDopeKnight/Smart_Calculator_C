1. В умном калькуляторе можно нажимать кнопки с клавиатуры(числа, операторы, скобки, вычисление(Enter), AC(del)), кроме функций, графиков и открытия других калькуляторов. Чтобы можно было вводить с клавиатуры, нужно ЛКМ нажать в область графика.
2. Кредитный и Депозитный калькуляторы могут производить расчет при нажатии клавиши Enter и имеют привязку к клавишам(кнопки подписаны) кредитный -F1, депозитный - F1/F2.
3. Чтобы установить калькулятор - make install, для удаления - make uninstall, создать архив с дистрибьютором - make dist, для удаления ненужных файлом - make clean
